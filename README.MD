### install
```
npm i fortnite-replay-parser (https://github.com/xNocken/replay-reader)
```

### run
```
node index.js
```

### nodejs to exe
```npm install -g pkg```
```pkg index.js -o "xdxReplayParser"```

### clone example
```
mkdir example
cd example
git clone https://github.com/xNocken/replay-reader-demo-project.git
```
